# ZodicogAI

ZodicogAI is a hybrid archetype intelligence platform that integrates symbolic zodiac classification with MBTI cognitive typology using deterministic trait modeling and LLM-based behavioral synthesis.

## Architecture

- Frontend: Next.js + Tailwind + Recharts
- Backend: FastAPI + Gemini API
- Modeling: Deterministic trait vectors + LLM synthesis
- Deployment: Vercel + Render

## v1 Features

- Zodiac sign engine
- MBTI typology selection
- Hybrid behavioral synthesis
- Compatibility mode
- Radar visualization dashboard
